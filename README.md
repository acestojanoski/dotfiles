### Useful commands

- Search vim:
    ```sh
    :grep -F|-R 'word' path_example: src/validators/*.js
    ```
- Show search results:
    ```sh
    :copen
    ```
- Go back to terminal:
    ```sh
    CTRL+z
    ```
- Go back to vim:
    ```sh
    fg
    ```
- Navigate trough windows:
    ```sh
    CTRL+w+[arrows]
    ```
